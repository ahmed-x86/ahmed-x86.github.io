About

Welcome to my personal Hyprland configuration (dotfiles). This setup focuses on productivity, aesthetics, and a clean workflow using the Catppuccin color palette.

Features

  • Fast & Lightweight Arch Linux setup
  • Hyprland Wayland compositor
  • Catppuccin themed environment
  • Advanced Waybar configuration
  • Smooth window animations

Installation

Online Install

bash -c "$(curl -fsSL https://raw.githubusercontent.com/ahmed-x86/hyprland_dotfiles/refs/heads/main/online-install.sh)"

Offline Install

git clone https://github.com/ahmed-x86/hyprland_dotfiles
cd hyprland_dotfiles

chmod +x *
./main.sh
