Installation

Online Install (Fastest)

bash -c "$(curl -fsSL https://raw.githubusercontent.com/ahmed-x86/hyprland_dotfiles/main/online-install.sh)"

Offline Install (Manual)

git clone https://github.com/ahmed-x86/hyprland_dotfiles
cd hyprland_dotfiles
chmod +x *
./main.sh
