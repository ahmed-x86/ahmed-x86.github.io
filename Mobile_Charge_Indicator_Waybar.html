📱 Mobile Charge Indicator for Waybar

A lightweight script that shows your phone battery in Waybar

← Back to Home GitHub Repo

About

A sleek and lightweight script that displays your mobile phone battery status directly in Waybar using KDE Connect.

Features

  • Auto detection of connected phone ID
  • Dynamic battery colors (Normal / Warning / Critical)
  • Charging indicator icon
  • Click module to ping your phone

Prerequisites

  • kdeconnect
  • waybar
  • Nerd Font (JetBrains Mono Nerd Font recommended)

Installation

Run the following commands in your terminal to install the script:

cd mobile_charge_indicator_waybar
chmod +x install.sh
./install.sh

The script will automatically copy mobile.sh to: 

~/.config/waybar/scripts/

Waybar Configuration

Add this module to your Waybar config:

"custom/mobile": {
    "format": "{text}",
    "exec": "~/.config/waybar/scripts/mobile.sh",
    "interval": 30,
    "return-type": "json",
    "on-click": "~/.config/waybar/scripts/mobile.sh --ping"
}

Waybar Style

Add this CSS to your Waybar style file:

#custom-mobile {
    font-family: "JetBrains Mono Nerd", monospace;
    font-size: 14px;
    padding: 0 6px;
    margin-left: 4px;
}

#custom-mobile.critical {
    color: #f07178;
}

#custom-mobile.warning {
    color: #f2cd66;
}

#custom-mobile.normal {
    color: #c3e88d;
}

Usage

Restart Waybar and your phone battery level will appear on the bar. Click the module anytime to ping your phone.

Notes

  • KDE Connect daemon must be running
  • Your phone must be paired
  • Tested on Arch Linux with Waybar
